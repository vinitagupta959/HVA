Heading Level 1

The H1 heading is used to indicate very important content. When creating the main heading for a page, we use H1 as it signifies the highest level of importance. The default font size for H1 is 32px.

Heading Level 2

The H2 heading is used for subheadings that are slightly less important than H1. Typically used for main sections, it has a default font size of 24px.

Heading Level 3

The H3 heading is used for sub-subheadings, which have less importance than H2. It is usually used within sections or subsections, with a default font size of 20.8px.

Heading Level 4

The H4 heading is used for additional sub-sections under H3. It is a smaller heading with a default font size of 16px.

Heading Level 5

The H5 heading is used for minor headings that are less important than H4. Its default font size is 12.8px.

Heading Level 6

The H6 heading is the smallest heading level, used for the least important headings. Its default font size is 11.2px.
